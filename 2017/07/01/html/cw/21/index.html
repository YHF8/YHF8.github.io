JavaScript正则表达式入门

发表于 2017-07-01 | 分类于 study_notes

正则表达式

正则表达式(regular expression)是一个描述字符模式的对象。ECMAScript的RegExp类表示正则表达式，而String和RegExp都定义了使用正则表达式进行强大的模式匹配和文本检索与替换的函数。

正则表达式主要用来验证客户端的输入数据。用户填写完表单单击按钮之后，表单就会被发送到服务器，在服务器端通常会用PHP、ASP.NET等服务器脚本对其进行进一步处理。因为客户端验证，可以节约大量的服务器端的系统资源，并且提供更好的用户体验。

创建正则表达式

RegExp 对象方法

test() 方法

test() 方法检索字符串中的指定值。返回值是 true 或 false。

exec() 方法

exec() 方法检索字符串中的指定值。返回值是被找到的值。如果没有发现匹配，则返回 null。

compile() 方法

compile() 方法用于改变 RegExp。

正则方法

search

search: 通过字符串或正则表达式匹配指定字符串的位置，如果没有找到任何匹配的字符串，则返回 -1。

match

match：找到一个或多个正则表达式的匹配

replace

replace：替换与正则表达式匹配的子串。

元字符

元字符 描述
. 查找任意字符，除了换行和行结束符。
\w 查找字母数字和下划线。
\d 查找数字。
\s 查找空白字符。
\b 匹配单词边界。
\u 匹配汉字。
^ 匹配字符串开头。
$ 匹配字符串结尾。
[abc] 查找方括号之间的任何字符。
[^abc] 查找任何不在方括号之间的字符。

逻辑分组

  • | : a|b|c 匹配 a 或 b 或 c 中任意一个
  • (…) : 分组表达式作为一个整体

量词

量词字符 描述
n+ 匹配任何包含至少一个 n 的字符串。
n* 匹配任何包含零个或多个 n 的字符串。
n? 匹配任何包含零个或一个 n 的字符串。
n{X} 匹配包含 X 个 n 的序列的字符串。
n{X,Y} 匹配包含 X 或 Y 个 n 的序列的字符串。

例子

  • 数字：\^[0-9]*$
  • 由数字，26个英文字母或下划线组成的字符串：^\w+$或者 \^[a-zA-Z0-9_]+$
  • 字符串只能是中文：\^[\u4e00-\u9fa5]{0,}$
  • 双字节字符：[^\x00-\xff]
  • 日期格式：^\d{4}-\d{1,2}-\d{1,2}
  • 一年的12个月(01～09和1～12)：^(0?[1-9]|1[0-2])$
  • 一个月的31天(01～09和1～31)：^((0?[1-9])|((1|2)[0-9])|30|31)$
  • 手机号码：^1[34578]\d{9}$
  • Email地址: ^(\w)+(.\w+)*@(\w)+((.\w+)+)$;
  • 密码(以字母开头，长度在6~18之间，只能包含字母、数字和下划线)：^[a-zA-Z]\w{5,17}$
  • 首尾空白字符的正则表达式：^\s|\s$
  • HTML标记的正则表达式 <[^>]*>
  • 判断IE的版本：^.MSIE [5-8](?:\.[0-9]+)?(?!.Trident\/[5-9]\.0).*$
  • 校验身份证号码15位:\^[1-9]\d{7}((0\d)|(1[0-2]))(([0|1|2]\d)|3[0-1])\d{3}$
  • 校验身份证号码18位:\^[1-9]\d{5}[1-9]\d{3}((0\d)|(1[0-2]))(([0|1|2]\d)|3[0-1])\d{3}([0-9]|X)$
