jQuery DOM操作

发表于 2017-07-04 | 分类于 study_notes

节点筛选

结构

eq(n)

获取第N个元素,从0开始，可以负数-1代表最后一个。

first()

获取第1个元素

last()

获取最后1个元素

filter()

筛选出与指定匹配的元素集合。

is()

根据选择器、DOM元素或 jQuery 对象来检测匹配元素集合，如果其中至少有一个元素符合这个给定的表达式就返回true。

map()

将一组元素转换成其他数组（不论是否是元素数组）

children()

获取所有子元素的元素集合。

find()

获取元素指定的后代元素

next()

获取元素的下一个同级元素

prev()

获取元素的上一个同级元素

parent()

获取元素的父级元素

替换节点

结构

replaceWith()

将所有匹配的元素替换成指定的HTML或DOM元素。

replaceAll()

用匹配的元素替换掉所有 selector匹配到的元素。

删除节点

结构

remove()

从DOM中删除所有匹配的元素。删除后返回删除的对象，但绑定的事件，附加的数据等都会被移除。

detach()

从DOM中删除所有匹配的元素。删除后返回删除的对象，但绑定的事件，附加的数据等都会保留下来。

empty()

删除匹配的元素集合中所有的子节点。

复制节点

结构

clone()

克隆匹配的DOM元素并且选中这些克隆的副本。布尔值参数（true 或者 false）指示事件处理函数是否会被复制。

属性操作

attr()

获取或设置元素指定的标签行内属性

removeAttr()

删除元素指定的标签行内属性

prop()

获取或设置JQ对象的属性值

removeProp()
