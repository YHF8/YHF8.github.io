BOM-浏览器对象模型

浏览器对象模型提供了独立于内容的、可以与浏览器窗口进行互动的对象结构。BOM由多个对象组成，其中代表浏览器窗口的Window对象是BOM的顶层对象，其他对象都是该对象的子对象。

window对象

所有浏览器都支持 window 对象。它表示浏览器窗口。所有 JavaScript 全局对象、函数以及变量均自动成为 window对象的成员。全局变量是 window 对象的属性。全局函数是 window 对象的方法

location对象

对象属性

属性 描述
href 返回完整的 URL
hash 返回一个URL的锚部分
search 返回一个URL的查询部分
hostname 返回URL的主机名
pathname 返回的URL路径名
port 返回端口号
protocol 返回一个URL协议

对象方法

方法 描述
reload() 重新载入当前文档
assign() 载入一个新的文档
replace() 用新的文档替换当前文档

属性 描述

方法 说明
back() 返回上一页
forward() 进入下一页

Navigator对象

对象属性

属性 描述
appCodeName 返回浏览器的代码名
